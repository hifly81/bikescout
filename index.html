Protocol Architecture

Bridging LLMs to Physical Terrain Reality

01. Data Ingestion

The MCP server hooks into OpenStreetMap, SRTM-V3, andother data sources. It doesn't just read maps; it extracts ground roughness, vegetation density, and historical drainage patterns.

  • GeoJSON Stream Parsing
  • Satellite Moisture Index
02. AI Reasoning

Via Model Context Protocol, LLMs like Claude or GPT-4 query the BikeScout server to calculate if an S3-grade trail is rideable after 15mm of rain based on soil composition.

  • Predictive Mud Logic
  • Battery Range Prediction
03. Tactical Output

Get a full tactical briefing: recommended tire pressure, wind exposure vectors, and a "Go/No-Go" status tailored to your specific gear and real-time weather.

  • Tire Pressure Calc
  • Exposure Risk Map
